# 案例录入模板

公开案例不得包含客户姓名、联系方式、账号、密钥、私聊截图、供应商底价或其他未经授权的信息。

```markdown
# 案例名称

- 日期：
- 行业与用户：
- 证据等级：A / B / C / D
- 是否已脱敏：是 / 否
- 是否允许公开：是 / 否
- 用户是否逐次授权进入知识候选：是 / 否

## 原问题

## 已知事实

## 当时的假设

## 执行动作

## 结果数据

- 触达：
- 有效对话：
- 报价：
- 付款或退款：
- 交付工时：
- 毛利：

## 最早错误或成功信号

## 适用边界

## 反例、失败条件与副作用

## 下一次会怎么做

## 可沉淀到哪个 Skill
```

默认只保存在本机状态。没有逐次授权时，不进入知识候选；单个案例即使成功，也不能直接升级成稳定方法。
