# 天策公开 X 内容索引

当前共有 **489 条** `@Leobai825` 公开原创内容，日期范围为 **2025-11-27 至 2026-07-14**。长推文已尽量用 X 归档中的 Note Tweet 正文补全。

## 它是什么

[`posts.jsonl`](posts.jsonl) 是历史公开内容的可搜索索引，用来回答“天策过去公开说过什么”。每行包含：推文 ID、日期、正文、点赞、转发、互动合计、原链接和状态。

## 它不是什么

- 不代表天策今天仍认可或仍在做原文中的事情；
- 不代表 TC 把每句话收录成方法；
- 互动数只是归档时快照，不是实时数据；
- 里面的收入、粉丝、价格、合作与业务状态都是历史自述，未经独立审计时不能用作收益承诺。

真正能被 `/tc` 当作候选方法使用的内容在 [`../原子库/atoms.jsonl`](../原子库/atoms.jsonl)，再经过筛选后进入 [`../Skill知识包/`](../Skill知识包)。

## 更新方法

只从账号所有者自己的 X 归档中导出公开原创内容：

```bash
python3 tools/import_x_archive.py \
  --archive /path/to/x-archive \
  --output 知识库/公开内容索引/posts.jsonl \
  --handle Leobai825
```

脚本只读取 `data/tweets.js` 和 `data/note-tweet.js`，不会读取私信、联系人、账号安全、广告画像或授权令牌。
