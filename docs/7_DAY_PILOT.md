# TC 七天内测招募与执行稿

## 招募文案

我最近把自己过去做自媒体、校园项目、AI 应用、社群和创业复盘的方法，做成了一个开源 Skill。

它不会给你一整套创业课。它会先帮你看见自己真正想问、真正需要解决的问题，再给出判断、取舍和一个能验证答案的动作。

现在招募 10 位内测用户，覆盖五种状态：

- 想创业但完全没方向
- 有想法但没验证
- 有产品但没流量
- 有人咨询但没成交
- 已经成交但交付很乱

内测持续 7 天。你每天只需要执行一个动作，并把真实结果带回来。我们会记录用户对话、报价、付款、交付和复购，但不会承诺你一定赚钱。

参与要求只有两个：愿意真实执行，愿意提供真实反馈。

## 首次使用

在同一个任务里输入：

```text
/tc 新手入门
```

不要提前整理完整商业计划。直接说你现在在做什么、卡在哪里，说得乱也可以。

Skill 给出动作后，先去做。完成后回到同一个任务，发送：

```text
我做完了，结果是：
用户原话：
实际数据：
我卡住的地方：
```

## 最终验收

七天后不问“你觉得这个 Skill 厉不厉害”，只检查：

- 有没有更准确地说出自己真正需要解决的问题。
- TC 的判断是否帮助用户减少无关动作或错误投入。
- 有没有开始行动。
- 有没有接触真实用户。
- 有没有提出报价。
- 有没有得到付款或更强的收入证据。
- 有没有愿意继续下一轮。
