# TC v1.3.0

v1.3.0 把 TC 从“回答很多内容的创业提示词”重构成一个渐进式解题系统。

## TC 是什么

用户把一件讲不清、想不明白或推不动的创业问题交给 `/tc`。TC 先重构和定义真正的问题，用户确认后，再给一个明确方案、必须接受的代价、第一步和有效标准。

## 这次改了什么

- `/tc` 成为普通用户唯一需要记住的入口；
- 只输入 `/tc` 时，用一句话接住用户，不再展示长菜单；
- 复杂问题拆成两轮：先定义问题，再给方案；
- 简单明确的问题直接回答，不强行“深挖”；
- 增加 `tc-diagnosis`、`tc-copy`、`tc-action` 和 `tc-update` 的清楚边界；
- 主入口增加 16 份按需读取的参考资料；
- 增加行为验收，防止长报告、空框架、收益承诺和无效行动。

## 最短用法

安装：

```bash
npx -y skills add Leobai03/tc -g --all
```

新开对话：

```text
/tc
```

也可以直接发送：

```text
/tc 我有流量、社群和几个项目，但不知道未来 90 天应该放大什么。
```

## 下载哪个文件

- `tc.zip`：只安装主入口；
- `tc-suite-1.3.0.zip`：完整 5 个 Skill；
- `tc-diagnosis.zip`、`tc-copy.zip`、`tc-action.zip`、`tc-update.zip`：单独安装对应能力。
