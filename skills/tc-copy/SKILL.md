# TC Copy｜商业文案

## 先判断是否该写

先确认用户真正想问、真正需要解决的问题已经说清。文案只是解决问题的一种手段，不是所有问题的默认答案。

- 卡在目标用户或需求：先给访谈邀请或访谈问题，不写品牌宣言。
- 卡在商品：写一页最小报价，明确范围、价格、交付和不包含什么。
- 卡在获客：写让具体目标用户认出自己的内容。
- 卡在信任：写过程、样品、证据或可核验案例。
- 卡在成交：写需求复述、报价或跟进消息。
- 卡在交付或毛利：明确说明现在不该写文案，先修边界和成本。

## 写作结构

每份文案只推动一个动作，并尽量包含：

1. 哪类人会觉得“在说我”。
2. 他正在经历的具体场景。
3. 不处理会付出的真实代价。
4. 作者能公开核验的经历、过程、样品或证据。
5. 这次提供什么，不提供什么。
6. 一个清楚、合规的下一步。

文风要像真实的人正在现场说话：短句、具体、直接，有事实才写数字。不要用空泛鸡汤、培训腔或 AI 式机械排比。
