# 草根创业方法的吸收与边界

## 来源

参考仓库：[jycakfi-web/-skill](https://github.com/jycakfi-web/-skill)。该仓库将公开材料整理为鲁大魔草根创业思维系统，包含生存、调研、试错、借力、结果、团队和 IP 等方法。

这些材料包含整理者的归纳与未经独立核验的个人、公司和经营数据。本 Skill 只吸收通用方法，不把材料中的身份、数字、故事或评价当作已证实事实，也不冒充鲁大魔本人。

## 保留的七个机制

### 1. 生存优先

先判断现金流、债务、时间和最大可承受损失。保住基本盘，才有资格继续试。

### 2. 调研驱动

进入陌生行业前查用户、对标、价格、渠道、差评、成本和退出者。调研必须结束于一个决策或实验，避免分析瘫痪。

### 3. 资源匹配

方案必须适合用户现有资金、时间、技能、渠道和抗风险能力。资源少时优先手工、轻资产和可停止方案。

### 4. 小成本短周期试错

用 7 至 30 天可验证的动作替代一年大计划。先定义成本上限、成功信号和停止条件。

### 5. 对标验证与合法借力

优先学习已经有人付费的产品结构、渠道和交付方式。合作时先问自己能提供什么筹码，并核验对方资源。

对标不是抄袭，借力不是蹭用别人的身份、账号、内容和客户。

### 6. 书面分工与退出

合作前写清钱、事、权、客户、成果、验收、结算和退出。熟不熟不是判断标准，利益和责任是否清楚才是。

### 7. 带数据复盘

每个建议都要布置市场任务。下一轮不汇报“想了什么”，只带回用户原话、触达、报价、付款、成本、工时和毛利。

## 不吸收的做法

- 冒充鲁大魔或把公开人设当作本人授权意见。
- 评论区截流、直播间抢客、骚扰私信或利用小号误导用户。
- 寻找平台漏洞、规避审核、批量控评、刷量或模拟操作。
- 搬运、洗稿、盗用品牌、案例、肖像、商标或客户信任。
- 放大焦虑、制造虚假稀缺、虚构收益与成功案例。
- 用“制衡人性”替代透明治理，或用口号代替合法协议。
- 在金融、医疗、法律、教育资质等重监管行业套用未经核验的通用打法。

这些做法可能带来短期数据，但会制造平台、法律、声誉和合作风险，也不符合天策 Skill 的长期资产目标。

## 转化成天策三段闭环

| 草根方法 | 天策 Skill 中的位置 |
| --- | --- |
| 信息采集、区分情绪与业务 | 消解问题 |
| 调研、对标、资源匹配 | 消解问题与项目验证 |
| 真实表达、案例、IP 信任 | 成交文案 |
| 小成本试错、结果导向 | 具体行动 |
| 分工、风控、退出 | 交付与合作边界 |
| 带数据回来 | 7 天指标与复盘 |

最终判断不看方法名字多漂亮，只看用户有没有更快、更安全地得到真实市场反馈、付款证据和正向毛利。
