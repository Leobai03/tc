# 跨平台安装与使用

## 核心原则

不同 AI 平台不能共享同一种“安装”方式。只给用户当前平台最短的路径：

- 原生支持 Agent Skills：安装完整 Skill 包。
- 支持自定义智能体或系统提示词：把轻量提示词放入智能体指令。
- 只有普通聊天框：复制轻量提示词作为新对话第一条消息。
- 微信用户：打开统一入口页，再选择下载或复制；不能声称微信会自动安装外部 Skill。

统一入口：`https://leobai825.icu/tc.html`

## 平台路由

### Codex 与支持 Agent Skills 的 ChatGPT

优先提供官方 GitHub 安装命令：

```bash
npx -y skills add Leobai03/tc -g --all
```

无法使用终端时，再提供完整 ZIP 包或 Skill 分享链接。安装后新开对话并输入：

```text
/tc 帮我把这个创业问题消解清楚，判断现在该写什么，并给我一个今天能完成的市场动作：
【粘贴自己的情况】
```

不同产品和端之间可能不会自动同步安装状态。界面与权限会变化时，引导用户查看该平台最新官方说明，不编造按钮位置。

### 豆包、DeepSeek、Kimi、元宝及普通聊天 AI

这些普通聊天界面不一定原生识别 Skill 文件。不要让小白研究目录和命令：

1. 打开一个新对话。
2. 复制 `assets/tc-lite.txt` 全文并发送。
3. 再发送自己的创业情况。

如果平台支持“智能体、角色、系统提示词或自定义助手”，把轻量提示词放进长期指令，只做一次。

### DeepSeek API 或其他兼容 API

开发者可以把轻量提示词作为 `system` 消息，用户的创业问题作为 `user` 消息。不要要求普通用户申请 API Key。

### 微信小程序

外部 Skill 不能靠一条聊天消息安装进任意微信小程序。最低门槛方案是把统一入口页分享到微信，让用户打开后复制轻量提示词。若要做到“点开就聊”，需要作者另外在智能体平台发布一个公开智能体或开发小程序。

## 小白使用模板

安装或复制完成后，只让用户填写五行：

```text
我现在卖什么：
我想卖给谁：
我已经做过什么：
现在有什么数据：
我最多能投入多少时间和钱：
```

不知道的可以写“不知道”。Skill 应继续消解，而不是拒绝回答。

```text
我做了一个创业 AI Skill，名字叫 TC。

它会帮你做三件事：把问题想清楚、写出真正需要的成交文案、给一个今天能执行的市场动作。

手机和电脑都能用。打开这个链接，选择你正在用的 AI：
https://leobai825.icu/tc.html

Codex / ChatGPT 可以下载完整 Skill；豆包、DeepSeek、Kimi 等工具直接复制轻量版提示词。装好以后只需输入 /tc。不会自动上传你的聊天记录。
```

GitHub 项目：`https://github.com/Leobai03/tc`

## 降低门槛检查

- 不要求普通用户会 Git、终端、API 或文件目录。
- 不一次展示所有平台教程，先让他选择正在使用的工具。
- 所有按钮使用“下载完整 Skill”“复制到豆包/DeepSeek”“复制提问模板”等结果语言。
- 明确安装和复制的区别，不假装所有平台原生支持。
- 手机页面必须能复制、下载、发邮件和访问作者网站。
